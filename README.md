# battery-curve-tool

1. 手机上安装power-record.apk，开启飞行模式，关闭自动背光，将电池充满电后进行放电记录放电曲线，或者将电池放空后进行充电记录充电曲线。
2. 数据记录在手机目录 /sdcard/powerrecord.txt
3. PowerFileParse为QT源码编译出来的工具用于解析/sdcard/powerrecord.txt绘制曲线
